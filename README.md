## Description
This project analyzes a sales dataset to practice data cleaning, analysis, and visualization using Python and pandas.

## Skills & Tools
- Python (pandas, matplotlib)
- Data cleaning: handling missing values, duplicates, and date formatting
- Data visualization: bar charts and line charts
- Extracting insights from data

## Steps
1. Load the sales data (`sales_data_dirty.xlsx`) into Python using pandas
2. Identify and handle missing values in 'Sales' and 'Category'
3. Remove duplicate rows
4. Convert 'Date' column to a uniform datetime format
5. Analyze total sales by category and over time
6. Visualize results using matplotlib

## Insights
- Electronics had the highest sales
- Home had the lowest sales (excluding Unknown)
- Sales increased over time
